# 1. Introduction

## 1.1 What is an e-Portfolio?

An **e-Portfolio** is a digital collection of a student's academic work, reflections, achievements, and learning progress.

In Moodle, the **Exabis ePortfolio** plugin allows students to:

* Upload files and documents
* Save web links
* Write notes and reflections
* Organize artifacts into folders
* Create portfolio pages (Views)
* Share work with teachers for feedback

This tool supports **continuous learning, reflection, and competency-based education**.

# 5.4 Creating Portfolio Views

A **Portfolio View** is a page that displays selected portfolio artifacts together in an organized format.
Instead of sharing the entire portfolio, students can create a **custom view** to present specific artifacts related to a course, project, or semester.

Portfolio views are commonly used to **showcase coursework, projects, and reflections**.

📷 **Screenshot Placeholder**
`[Screenshot: My Views section in Exabis e-Portfolio]`

---

### Example Scenario

Program: **BSc in IT**
Semester: **Semester 01**

A student creates a portfolio view containing selected coursework.

Example portfolio view:

```
BSc IT – Semester 01 Portfolio

• Programming_Assignment_01.pdf
• Semester Reflection Note
```

---

### Steps to Create a Portfolio View

1. Open **Exabis e-Portfolio**.
2. Click **My Views**.
3. Click **Create New View**.

Enter the following details:

```
Title: BSc IT – Semester 01 Portfolio
Description: Portfolio containing assignments completed during Semester 01.
```

4. Select artifacts from the **Assignments category**.

Example:

```
Programming_Assignment_01.pdf
```

5. Arrange the artifacts.
6. Click **Save**.

# 5.5 Sharing Portfolio Views

After creating a portfolio view, students can share it with their teacher.

This allows the teacher to review the student’s work and provide feedback.

---

### Steps to Share a Portfolio View

1. Open **My Views**.
2. Select the view:

```
BSc IT – Semester 01 Portfolio
```

3. Click **Share**.
4. Select the **course or teacher**.
5. Enable comments (optional).
6. Click **Save**.
